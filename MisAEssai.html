Les tests du programme

Test 1: Dans le programme d'addition 4-6, j'ai entré une lettre pour voir si ça me donnais une erreur. J'attendais que ça afficherais une erreur. Par contre, le programme a simplement dit que ce n'était pas la bonne réponse.

Test 2: Dans le programme d'addition 4-6, j'ai entré un symbole moins (-) pour voir si ça me donnait une erreur. J'attendais que ça afficherais une erreur. Par contre, le programme a simplement dit que ce n'était pas la bonne réponse.

Test 3: Dans le programme de division 7-8, j'ai fais plusieurs essais pour voir si le quotient des deux nombres au hazard était un nombre décimal. Je pensais qu'il y allait avoir des quotients décimaux, mais cela n'est pas arrivé.

Test 4: Après avoir écrit le code des trois pages, je suis allé voir si les changements faits pour les difficultés des questions dépendant des années fonctionnait. Puisque dans certains des cas les questions étaient soit trop faciles ou difficiles, nous avons modifié les fonctions des opérations pour qu'ils soient associés à leur niveau scolaire avec plus de précision.

Test 5: Comme test final, nous avons laissé un membre d'une autre équipe du groupe classe essayer notre programme et nous donner du "feedback" sur ce qu'il croit peut être meilleur. D'après lui, notre programme est bien fait pour les années choisies et est très bon comme il l'est.
